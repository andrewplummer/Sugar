setItem

Boolean setItem ( data , location )
Saves data to local storage. It returns a String that can be one of three values: "true" if the storage succeeded; "false" if the user has denied storage on their machine or storage space allotted is not sufficient.

The size limit for the passed parameters is ~40Kb.

Parameters:
data <Object> The data to store
location <String> The name of the "cookie" or store
Returns: Boolean
Whether or not the save was successful
