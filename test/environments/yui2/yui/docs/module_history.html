Module: history

The Browser History Manager provides the ability to use the back/forward navigation buttons in a DHTML application. It also allows a DHTML application to be bookmarked in a specific state. This library requires the following static markup: <iframe id="yui-history-iframe" src="path-to-real-asset-in-same-domain"></iframe> <input id="yui-history-field" type="hidden">
